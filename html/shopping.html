Buying your Arduino

There are many different Arduino and Arduino-compatible boards. You'll need one which can communicate as a Serial USB device which excludes some of the smaller cheaper boards (but those can still work great for other projects, like simulating a USB keyboard).

To keep things simple, the Arduino Uno is an excellent board for getting started with. It's a little bit big, but that can help if you're new at all of this because everything is labelled clearly on the device. It also offers some over-voltage short circuit protection from damaging the board or your computer (though you may still kill some small components like LEDs as you get started by hooking them up incorrectly, but that's ok).

http://arduino.cc/en/Main/ArduinoBoardUno

It should cost about $25.

https://www.adafruit.com/category/17

https://www.sparkfun.com/categories/103

http://floris.cc/shop/en/6-arduino-

Another great board that will work just fine is the Teensy. There are a few versions available, but the only one I've tested is version 3.1. I'll add support for the other boards, as they should work as well. Think of it as a smaller, faster, better Arduino-compatible board that's great for you if you're comfortable with a little more soldering.

https://www.pjrc.com/teensy/

Components

Just browsing for components can give you a lot of ideas:

https://www.adafruit.com/category/54

https://www.sparkfun.com/categories/51

...but there are a few things to look out for when buying components. Just because it's a component or a sensor that can work with the Arduino doesn't mean that it will work with the Control Board extension, as some components require some extra programming or extra use of libraries on the Arduino itself, but we won't be getting into that. When in doubt keep it simple, but here are some other things to look out for:

Buttons and Switches:

If the description of a button or a keypad says that it needs to do any special tricks, like to have the micro controller scan through the columns and rows of the keypad (https://www.adafruit.com/products/1824) or if it uses the “I2C bus” (https://www.adafruit.com/products/1616) then they won't work with the Control Board extension as it is right now. Stick with simple components that wire up each button on their own, like basically everything here:

https://www.adafruit.com/categories/235

Buttons and switches always need resistors, at least for a very easy way to get them working in a basic circuit with Control Board. They come in a wide range of resistance values, but a 10k-ohm resistor should do it. Resistors can be very cheap when purchased from the right place, so you might want to buy a pack that has a lot of values.

LEDs

Again, nothing too fancy. Adafruit has a great line of “Neopixel” products, which are RGB LEDs in strips and arrays, but they won't work here, at least not how Control Board currently works. Keep it simple and stick to basic bare LEDs: https://www.adafruit.com/categories/90

When shopping for an RGB LED there are two kinds: “Common-Cathode” or “Common-Anode”. With the simplest way of connecting them and using them with Control Board they'll both work but the Common-Anode variety would end up with the opposite range of values when adjusting the brightness. Stick with Common-Cathode for now (I'll explain more in a circuit example elsewhere)

LEDs always need resistors. To really choose the proper resistor value to match up with a particular LED can be tricky at first, but if you don't care about dimming your LED too much (which is fine, they're usually too bright), then somewhere between 300-ohm and 1k-ohm should be just fine.

Potentiometers

A potentiometer is a variable resistor, which you would connect as an Analog Sensor to your Arduino to be able to sweep through a range of values from 0 to 1.

They come in a range of resistance values which won't matter too much with how you're using them (perhaps look for something around 10k-ohm). But, the main thing to look out for is that there are 3 types of potentiometers: A, B, and C. The “B” type will have a “Linear” response, which means that half way through the turn the Arduino will be reading 0.5, which is probably what you want. The “A” type has a logarithmic curve which is more suited for audio applications, and the “C” type is a reverse-log. So, a “B10k” or even a “B100k” potentiometer would be great.

If you're using a Breadboard to prototype your circuit (more on that in the next section) you'll also want to make sure that you're getting a potentiometer that can be mounted on a board. Take for example these two potentiometers:

https://www.adafruit.com/products/562

http://www.taydaelectronics.com/potentiometer-variable-resistors/rotary-potentiometer/linear/1k-ohm-linear-taper-potentiometer-round-shaft-solder-lugs-3574.html

The first one has thin pins that will mount in your board, the second one is designed for wires to be soldered to it.

Prototyping boards and jumper wires

You'll very likely want a prototyping Breadboard to get started with, which will keep you from having to do any soldering straight away, along with some jumper wires to make connections between the Arduino and your components.

Some starter kits are also available that have a good range of components and the prototyping boards that you would need, along with the Arduino Uno itself, and could be a great choice for the beginner. As with any kit you run the risk that you're buying components you might not want (motors? battery packs?) but it might be a way to get a little bit of everything:

https://www.adafruit.com/products/193

https://www.adafruit.com/products/68

https://www.adafruit.com/products/170

Motors

If you want to go crazy and add motors to your project, just watch out that most motors need to draw more power than your little Arduino can give, which would require a slightly more complex circuit and another power adapter. However, small motors can work just fine.

A micro servo motor like this could work without any trouble:

https://www.adafruit.com/products/169
